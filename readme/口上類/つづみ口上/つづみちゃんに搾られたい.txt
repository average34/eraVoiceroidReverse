◎登場人物
・あなた
哀れな家畜。マゾ。
そもそもこのバリアントで遊ぶ人だいたいマゾでしょ
放出する精液は貴重なエネルギー源となる。つづみはこれを回収し魔力として利用している。

・すずきつづみ
搾精に特化したセクサロイド。技能が初めからついている。
妊娠することは無い。それどころか精子を体内に貯蔵し自由に取り出せるようにできている。
感情をあまり表に出さず、無表情で責め続けてくる。
とはいえ感情そのものが無いわけではなく、調教やセックスをわりと楽しんでいる。
調教中はあなたの精液を出させることしか考えていないが、それが普通だと思っている。仕事だし。

CeVIOの中でも性的能力を特化させた存在。「さとうささら」は男性の性処理を行うのを目的としているが、「すずきつづみ」は業務用ザーメンの搾精に特化している。
この施設で使われている「すずきつづみ」もまたそのパッケージの一つ。同型が施設に20体くらいはいる。
すずきつづみ自体が結構なお値段するのだが、その精液エネルギー変換によって得られるエネルギー量は火力発電所の数倍なので、プラントを建てるよりお得。

手コキ、脚コキがうまい。性交の技能はそこそこある。
