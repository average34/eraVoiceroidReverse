口上補正
	CALL SET_KOJO_CORRECTION_T, コマンド名あるいはコマンド番号, メニュー名あるいはメニュー番号、補正値、対象のキャラ番号

	で、補正値を追加できます
	メニューは加虐における足コキと愛撫における足コキに別に補正を与えたい場合などに利用してください
	なにもかかなければすべての足コキに補正値が、書いてあれば書いてあるやつだけに補正値がはいります
	※両方あると両方足すので注意
	
	なお補正値は変動補正値なのでRAND:Xの形で増えます
