;eratohoReverse最初の説明
;────────────────────────────────────
まずはこのバリアントに興味を示して、ダウンロードしてくれてありがとうございます。

このバリアントはタイトル通り、プレイヤーが調教されるゲームです。
今までの逆調教とは違って、プレイヤーに調教の主導権がまったくありません。
ただ調教者が選択した行動に反応するだけのゲームです。

たとえばあなたは幽香に自慰してと命令されました。
まだ従順になってないあなたは命令を従ってもいいし、拒否してもいいです。
しかしここで拒否を選んだら幽香の不興を買うこととなり、
お仕置きとして鞭であなたを打つかもしれません。
鞭で打たれたら悲鳴を上げるか我慢するかの選択も出ますが、
痛みが強すぎる場合は叫ぶしかありません...

大体こんな感じのバリアントです。
自由性はかなり低いが、反応で調教者の行動を影響することはできますし、
下手な調教者の調教をもっとスムーズに誘導するか、
上手な調教者から脱走や逆襲を目指してもいいです。

いろんなバリアントを参考してますが、本来のeraとシステム的差異が多すぎるため、
ほぼ全部自分で一から書き込みました。そんな訳でプレイする前にこの三点はご了承してください：

（１）バグや設計のミスがたくさんあるかもしれません。
　　　致命的なバグでなくても、何か変な動きや理解できない状況が発生したら、
　　　遠慮なくスレやＩＲＣに報告してください。問題の解決に努力します
　　　もし自分でバグについで調べて直してくれるなら、最高の感謝を致します。

（２）システム的に大きく変更しましたから、ぜひ説明書を見てからプレイしてください。
　　　DATAフォルダに他にもゲームシステムの説明やまとめの文書を用意してるから、
　　　システムの仕組みについて疑問がある場合はそれらを見てくださればうれしいです。
　　　（DATAフォルダに入ってる文書は追記で詳しく説明します。）

　　　開発できたばかりで内容は豊富とは言えないし、設計的に思慮が足りない部分もあるだろうから、
　　　何か要望や意見があったらぜひスレやＩＲＣに書き込んで検討させてください。
　　　といいますが、これからは忙しくなるから、
　　　おそらく自分では次の例大祭が終わるまで大きな補足や改変は作れません。
　　　未完成の反抗準備、助手、触手、コスプレの処理もそれまで放置するようにします。
　　　バグの修正ならつくつく頑張りますが。

　　　そういうことで、追加機能として新しい行動や反応などを作ってくれるなら大歓迎です。
　　　勝手な調整や変更なども構いません。

（３）日本語はそれなりに自信がありますが所詮外人です。
　　　言語的に間違ってるところが多いと思いますし、
　　　文化的にも日本と異なる趣向がありそうです。もしかしたら日本の方々に不適切な表現もあるかもしれません。
　　　もしより正しい日本語に修正してくれるなら幸いです。

（１）知ってる人はかなり少ないと思いますが、このバリアントの元ネタは
　　　「禁忌2～Hospital Taboo～」という古い調教ゲームです。
　　　あのゲームでもっとも特別な設計は、調教対象の加虐心を煽るような調教を重ねたらなんと、
　　　調教開始の時に確率で「調教対象がプレイヤーを押し倒し、そのまま調教します」。
　　　そのモードに入ると調教メニューはヒロインが勝手に決めて、
　　　プレイヤーは我慢とか受け入れるとか限られた選択肢の中から選ぶしかありません。
　　　ゲーム自体にはいろいろ突っ込みたい所がありますが、
　　　あれこそ逆調教ゲームのあり方だと、青かった私の心に深く残しました。
　　　会話をひとつの調教分野として扱うのも、脱衣がとてもしんどいのもあのゲームの影響です。

（２）最近はＶＨというジャンルを知ってはまっています。
　　　逆調教バリアントを作るぐらいだしね、女主人公が犯されるのが好きもとても自然なことですね。
　　　EramakerでＶＨゲームを作るのも不可能じゃないですかね...

（３）こんな需要の低いやつを二ヶ月かけて作るとは...
