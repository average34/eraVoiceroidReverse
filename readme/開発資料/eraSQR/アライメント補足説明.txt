*アライメントとは
調教アクションを決定する際に用いられる要素の一つで、アライメントとが高いとご褒美がもらえ、低いとご褒美をもらえます。ん？言い換えると、アライメントが高いと快楽系の調教が多くなり、低いとSM系の調教が多くなります。

*アライメント変動の流れ

平時アライメントを導入し、調教の開始時にアライメント = 平時アライメントとする。

調教対象の反応に応じてアライメント変動の基本値を設定、腰をふるなら+大に、嫌がるなら-小に等々。

ランダム素質の決定に際して、SM軸性格(0 ～100で変動、100でドＳ)、年長-年少軸、ツン-デレ軸、淫乱-貞淑軸というのを用いているので、これをCFLAGで保存し、例えばドＳならアライメントの-方向への変動幅を大きくするという処理を加える。ランダム素質を設定しないキャラや、以前のバージョンのセーブデータで生成されたキャラは平均的な変動をします。

現在のアライメントが平時アライメントから離れるにつれて、変動幅が小さくなるように。

アライメントが極端な値の時、より極端な方向への変動が小さくなるように補正がかかります。

調教終了後、現在のアライメントに応じて平時アライメントが少し変動します。

*旧バージョンでは調教者側の要素に依存する部分が多かったのですが、本バージョンでは＋か－かの変動は調教対象の反応のみに依存し、変動幅に調教対象の要素が絡んでくるという仕様となっております。
